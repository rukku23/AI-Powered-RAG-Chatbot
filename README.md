# 🤖 AI-Powered RAG Chatbot

An intelligent Retrieval-Augmented Generation (RAG) chatbot that allows users to upload PDF documents and ask questions in natural language. The chatbot retrieves relevant information from uploaded documents and generates accurate, context-aware responses using Large Language Models (LLMs).

##  Features

- 📄 Upload and process PDF documents
- 🔍 Semantic search and context retrieval
- 🤖 AI-powered question answering
- ⚡ Fast and relevant document retrieval
- 💬 Interactive chat interface
- 📚 Document-based knowledge assistant

## 🏗️ Architecture

User Query
↓
PDF Processing
↓
Text Extraction
↓
Vector Embeddings
↓
Semantic Retrieval
↓
LLM Response Generation
↓
Answer to User

## ⚙️ Installation

### Backend

```bash
cd backend
pip install -r requirements.txt
python main.py
```

### Frontend

```bash
cd frontend
npm install
npm start
```
